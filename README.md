# LinkedIn Job Market Analytics

# About the Project
This project analyzes LinkedIn job posting data to understand hiring trends, salary distribution, required skills, and work arrangements. The aim of this project was to practice data analysis using SQL and present the findings through an interactive dashboard.

#Features
* Analyzed hiring trends across different companies and industries
* Compared salary distribution for different job roles
* Studied remote, hybrid, and onsite job opportunities
* Explored the most in-demand technical skills
* Created an interactive dashboard to visualize the analysis

#Tools & Technologies
* SQL
* Python
* HTML Dashboard
* Git & GitHub

#Project Files
* analysis.sql – SQL queries used for data analysis
* linkedin_jobs.csv – Dataset used in the project
* index.html – Interactive dashboard
* Business Insights.pdf – Summary of key findings
